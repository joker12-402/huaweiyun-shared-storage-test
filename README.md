# 微企业虚拟共享存储系统设计与验证
## TL;DR
- 内容：华为云 ECS + SFS/SFS Turbo 共享存储测试验证
- 覆盖：NFS/CIFS 挂载联调、多客户端并发、fio 性能压测矩阵、一致性检查、DNS 排障复盘
- 产出：脚本（scripts/）+ 测试记录（notes/）+ 报告/数据（test_report/）
## 项目背景
本科毕业设计项目，基于华为云为微企业设计轻量级虚拟共享存储解决方案，解决微企业存储成本高、跨平台共享难的问题（对应论文摘要/第一章）。
## 技术栈
华为云ECS/SFS/SFS Turbo、NFS/CIFS、fio、CentOS 7.4、Windows Server、云监控
## 核心工作
1. 搭建华为云ECS+SFS/SFS Turbo环境，实现Windows/Linux跨平台挂载与读写联调；
2. 解决nslookup域名解析失败问题，沉淀DNS配置排障步骤；
3. 使用fio完成顺序/随机读写的IOPS、带宽双维度性能压测；
4. 基于两台ECS验证多客户端共享存储的读写一致性，结合云监控分析压测指标。
## 项目成果
实现微企业级跨平台虚拟共享存储，完成性能与功能测试，形成可复现的测试报告与操作手册。
